# IT-AUL Landing Page

Инновационный адаптивный лендинг для команды разработчиков IT-AUL с уникальным пиксельным дизайном.

## 🚀 Особенности

- 🎮 **Pixel Art Design** - стильные пиксельные иконки вместо эмодзи
- 🌟 **Плавающие элементы** - динамические пиксельные декорации с анимациями
- 📱 **Адаптивный дизайн** - отлично выглядит на всех устройствах
- ⚡ **Webpack сборка** - оптимизация изображений, минификация кода
- 🎯 **Уникальные анимации** - каждый элемент имеет свою анимацию
- 🔮 **Glitch эффекты** - киберпанк атмосфера для заголовков
- 🎨 **Интерактивные hover эффекты** - трансформации и свечения
- 📊 **Анимированная статистика** - счетчики достижений
- 🌐 **Высокая производительность** - оптимизированные изображения и код
- 📈 **SEO-оптимизация** - правильная структура и мета-теги

## 🎨 Дизайн

### Пиксельный стиль
- 🎮 **Pixel Art иконки**: Качественные SVG пиксельные иллюстрации (ракета, самолет, часы, сердечко, динозавр)
- ✨ **Плавающие элементы**: Анимированные пиксельные декорации по всему сайту
- 🔮 **Glitch эффекты**: Киберпанк анимации для заголовков
- 🎯 **Уникальные анимации**: Каждый элемент имеет свою траекторию движения

### Цветовая палитра
- **Основной**: `#DDD92A` (желтый - фирменный цвет)
- **Вторичный**: `#000000` (черный)
- **Акцентный**: `#FFED4E` (светло-желтый)
- **Пиксельные эффекты**: Свечения и тени в желтых тонах

### Шрифты
- **Заголовки**: Unbounded (Google Fonts) - современный геометричный
- **Основной текст**: Unbounded - для единообразия
- **Специальные**: Lexend Exa - для технических элементов

## 🛠 Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/it-aul/landing.git
   cd landing
   ```

2. **Установите зависимости**
   ```bash
   npm install
   ```

3. **Команды разработки**
   
   **Для разработки (с hot reload):**
   ```bash
   npm run dev
   ```
   Откроется http://localhost:3000 с автоматической перезагрузкой

   **Продакшн сборка:**
   ```bash
   npm run build
   ```
   Создает оптимизированную версию в папке `dist/`

   **Анализ размера бундла:**
   ```bash
   npm run build:analyze
   ```
   Показывает детальную статистику размеров файлов

   **Предпросмотр продакшн версии:**
   ```bash
   npm run serve
   ```
   Запускает локальный сервер для тестирования собранной версии

4. **Альтернативный запуск (для старой версии)**
   - Откройте `dist/index.html` в любом современном браузере
   - Или используйте простой сервер:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx serve dist
   ```

## 📝 Управление достижениями

Достижения можно добавлять двумя способами:

### 1. Через веб-интерфейс
- Прокрутите до раздела "Наши достижения"
- Заполните форму "Добавить новое достижение"
- Нажмите "Добавить"

### 2. Через код
Добавьте новое достижение в `script.js` в методе `loadAchievements()`:

```javascript
{
  id: 5,
  title: "Новое достижение",
  description: "Описание достижения",
  date: "2024-04-01"
}
```

## 🎯 SEO и производительность

### Оптимизация изображений
- Используйте современные форматы (WebP, AVIF)
- Сжимайте изображения перед загрузкой
- Добавляйте alt-атрибуты

### Meta-теги
Измените meta-теги в `<head>` секции `index.html`:

```html
<title>Ваш новый заголовок</title>
<meta name="description" content="Ваше новое описание">
```

## 🎨 Кастомизация

### Изменение цветов
Измените CSS переменные в начале `styles.css`:

```css
:root {
    --primary-color: #YOUR_COLOR;
    --secondary-color: #YOUR_COLOR;
    --accent-color: #YOUR_COLOR;
}
```

### Добавление новых шрифтов
1. Добавьте ссылку в `<head>` секцию
2. Обновите CSS переменные шрифтов

### Изменение анимаций
Найдите и измените keyframes в `styles.css` или добавьте новые в `script.js`.

## 🐛 Решение проблем

### Шрифты не загружаются
- Проверьте подключение к интернету
- Убедитесь, что ссылки на Google Fonts корректны

### Анимации работают медленно
- Проверьте производительность браузера
- Отключите тяжелые эффекты в `script.js`

### Мобильное меню не работает
- Проверьте консоль браузера на ошибки JavaScript
- Убедитесь, что все скрипты загружены
