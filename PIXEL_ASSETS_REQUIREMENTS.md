# Pixel Art Assets Requirements для IT-AUL Landing

## 🎯 Общий стиль и концепция

**Основной стиль**: Минималистичный pixel art в стиле 8-16 бит с современным подходом
**Цветовая палитра**: 
- Основной: `#DDD92A` (яркий желтый)
- Акцентный: `#FFED4E` (светло-желтый) 
- Темный: `#000000` (черный)
- Дополнительный: `#333333` (темно-серый)
- Белый: `#FFFFFF` для контраста

**Технические требования**:
- Формат: SVG 
- Разрешение: 24x24px базовый размер (векторный, масштабируемый)
- Стиль: Четкие пиксельные границы, без сглаживания
- Изометрический уклон приветствуется для 3D эффекта

## 🎨 Recraft AI Prompts

### Базовый промпт для всех иконок:
```
Create a pixel art icon in 8-bit/16-bit style, 24x24 pixels, using primarily #DDD92A (bright yellow) and #000000 (black) colors. The icon should have clean pixel boundaries, no anti-aliasing, and maintain a minimalist design. Style should be reminiscent of classic video game graphics but with modern clarity.
```

### Специфичные промпты:

#### 1. React Logo Pixel Art
```
Create a pixel art version of React logo, 24x24px, using #DDD92A yellow and black colors. Simplify the atomic symbol to clear pixel boundaries, maintain the circular electron paths but in pixelated form. Clean 8-bit style, no gradients.
```

#### 2. Hackathon Trophy
```
Design a pixel art trophy for hackathon winners, 24x24px, bright yellow (#DDD92A) and black. Should convey achievement and technology. Simple geometric shapes, clear pixel boundaries, reminiscent of classic gaming trophies.
```

#### 3. Telegram Icon Pixel
```
Create a pixelated version of Telegram logo, 24x24px, using yellow (#DDD92A) paper airplane on black background. Maintain the iconic airplane shape but with clear pixel boundaries, 8-bit gaming style.
```

#### 4. Code Editor/Terminal
```
Design a pixel art code editor or terminal window, 24x24px, yellow and black color scheme. Show simplified window with text lines, maintain classic computer terminal aesthetic in pixel art style.
```

#### 5. Team Collaboration
```
Create pixel art showing team collaboration, 24x24px, yellow and black. Simple pixelated human figures connected or working together, abstract but recognizable, classic 8-bit game character style.
```

#### 6. Innovation Lightbulb
```
Design a pixel art lightbulb representing innovation, 24x24px, bright yellow (#DDD92A) bulb with black outline. Classic lightbulb shape but pixelated, maybe with small yellow "spark" pixels around it.
```

#### 7. Smartphone Pixel Art
```
Create a pixel art smartphone, 24x24px, black device outline with yellow screen elements. Simple rectangular shape with rounded corners in pixel form, show screen with basic UI elements.
```

#### 8. Database Icon
```
Design a pixel art database icon, 24x24px, yellow and black. Traditional cylinder stack representation but in clear pixel boundaries, 8-bit style data storage symbol.
```
