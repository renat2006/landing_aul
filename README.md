# IT-AUL Landing Page

Современный адаптивный лендинг для команды разработчиков IT-AUL.

## 🚀 Особенности

- **Адаптивный дизайн** - отлично выглядит на всех устройствах
- **Современные анимации** - плавные переходы и интерактивные элементы
- **Фирменный стиль** - использование корпоративных цветов и шрифтов
- **Легкое управление контентом** - JSON-файлы для быстрого обновления информации
- **Высокая производительность** - оптимизированный код и изображения
- **SEO-оптимизация** - правильная структура и мета-теги

## 🛠 Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/it-aul/landing.git
   cd landing
   ```

2. **Откройте в браузере**
   - Просто откройте `index.html` в любом современном браузере
   - Или используйте локальный сервер для разработки:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```

3. **Перейдите по адресу**
   ```
   http://localhost:8000
   ```

## ⚙️ Настройка контента

Основная информация хранится в файле `data.json`. Для изменения:

3. Внесите изменения

## 📝 Управление достижениями

Достижения можно добавлять двумя способами:

### 1. Через веб-интерфейс
- Прокрутите до раздела "Наши достижения"
- Заполните форму "Добавить новое достижение"
- Нажмите "Добавить"

### 2. Через код
Добавьте новое достижение в `script.js` в методе `loadAchievements()`:

```javascript
{
  id: 5,
  title: "Новое достижение",
  description: "Описание достижения",
  date: "2024-04-01"
}
```

## 🎯 SEO и производительность

### Оптимизация изображений
- Используйте современные форматы (WebP, AVIF)
- Сжимайте изображения перед загрузкой
- Добавляйте alt-атрибуты

### Meta-теги
Измените meta-теги в `<head>` секции `index.html`:

```html
<title>Ваш новый заголовок</title>
<meta name="description" content="Ваше новое описание">
```

## 🚀 Деплой

### GitHub Pages
1. Загрузите файлы в GitHub репозиторий
2. Включите GitHub Pages в настройках репозитория
3. Выберите ветку `main` как источник

### Netlify
1. Создайте аккаунт на Netlify
2. Подключите GitHub репозиторий
3. Настройте автоматический деплой

### Vercel
1. Установите Vercel CLI: `npm i -g vercel`
2. Выполните: `vercel`
3. Следуйте инструкциям

## 🎨 Кастомизация

### Изменение цветов
Измените CSS переменные в начале `styles.css`:

```css
:root {
    --primary-color: #YOUR_COLOR;
    --secondary-color: #YOUR_COLOR;
    --accent-color: #YOUR_COLOR;
}
```

### Добавление новых шрифтов
1. Добавьте ссылку в `<head>` секцию
2. Обновите CSS переменные шрифтов

### Изменение анимаций
Найдите и измените keyframes в `styles.css` или добавьте новые в `script.js`.

## 🐛 Решение проблем

### Шрифты не загружаются
- Проверьте подключение к интернету
- Убедитесь, что ссылки на Google Fonts корректны

### Анимации работают медленно
- Проверьте производительность браузера
- Отключите тяжелые эффекты в `script.js`

### Мобильное меню не работает
- Проверьте консоль браузера на ошибки JavaScript
- Убедитесь, что все скрипты загружены
